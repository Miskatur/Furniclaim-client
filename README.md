# FurniClaim Reseller Website

## Live Link 
Website : https://furniclaim.web.app/
### Features

 * Its a Furniture reseller website
 * Fully Responsive
 * Used ReactJS, TailwindCSS, DaisyUI, MongoDB, JWT and Environment Variables
 * MongoDB CRUD operation implemented
 * Separate  Buyer, Seller, Admin Dashboard 
 * Option to verify Seller mannually
 * Option to Pay
